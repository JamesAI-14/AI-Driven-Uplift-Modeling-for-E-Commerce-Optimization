# AI-Driven-Uplift-Modeling-for-E-Commerce-Optimization
Built a customer dataset from 100k+ transaction records and developed a T-Learner Uplift Model using dual Gradient Boosting Classifiers to predict coupon effectiveness. Simulated A/B tests and financial outcomes, increasing profits and reducing marketing waste by targeting persuadable customers.

Dataset found at: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
